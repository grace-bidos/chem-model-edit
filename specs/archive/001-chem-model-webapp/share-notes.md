# 共有方式メモ

## 現状

- 単一HTMLエクスポートのみ提供
- 3D表示は NGL の CDN を使用（オンライン必須）

## 目標

- 「リンク + サーバ保存」方式へ移行

## 想定アーキテクチャ

1. フロントで構造 + 表示状態を JSON 化
2. API `/share` に POST
3. サーバで保存し `share_id` を返す
4. `/share/:id` で読み込み + 表示再現

## 検討事項

- サイズ制限（大規模構造）
- 保存期限（デフォルトTTL）
- 公開/非公開トークン
- 共有ビューの編集可否

## 移行ステップ案

1. 現行 HTML エクスポート維持
2. API 保存方式を追加（DB or object storage）
3. 共有ページを実装しURL配布
